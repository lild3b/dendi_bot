# 📊 Personal Trading Performance Discord Bot

A private Discord bot built for personal trading discipline, performance tracking, and strategy execution improvement. Designed to help me stay consistent, track mistakes, and analyze trading behavior over time.

---

## 🚀 Features

### 📈 PnL Tracking System
- Daily profit/loss logging
- Trade count tracking
- Monthly performance calendar visualization
- Personal performance summaries

### 🧠 AI Trade Analysis
- Summarizes my trading activity from Discord messages
- Identifies repeated mistakes and habits
- Highlights winning patterns and execution quality
- Provides behavioral insights based on my trades

### ☑️ Pre-Trade Checklist System
A personal execution filter before entering trades:
- Liquidity sweep confirmation
- Market structure shift (MSS)
- Retest (OB / FVG)
- Higher timeframe alignment
- News filter check
- Anti-FOMO validation

Ensures I only take high-quality setups based on my rules.

---

## 🛠️ Commands

- `/checklist` → Pre-trade execution checklist (trade filter)
- `/pnl` → View my monthly performance calendar
- `/addpnl` → Log a trade result for a specific day
- `/pnlsum` → View my overall performance summary
- `/summarize` → AI summary of my recent trading activity

---

## 🎯 Purpose

This bot is built strictly for **personal use** to:

- Improve trading discipline
- Track emotional and execution mistakes
- Enforce rule-based trading
- Measure consistency over time
- Identify what actually works in my strategy
